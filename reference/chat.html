Jetpack Compose 中怎么实现一个带动画的 LazyColumn 列表项？
10:33
AI
可以用 animateItemPlacement() 修饰符，结合 LazyColumn 的 key 参数： 
// 每个 item 指定唯一 key
LazyColumn {
    items(
        items = list,
        key = { it.id }
    ) { item ->
        Text(
            text = item.title,
            modifier = Modifier
                .animateItemPlacement()
        )
    }
}
这样当列表项顺序改变时，Compose 会自动处理位置动画，无需手动管理。
